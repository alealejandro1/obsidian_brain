# Aggregate
Takes many rows and perform operations on them that return a scalar:
* AVG
* COUNT
* FIRST (not in [[MySQL]])
* LAST (not in [[MySQL]])
* MAX

# Scalar
Take 1 or 0 scalar inputs and return a value
* ROUND
* LEN
* NOW (requires no inputs)
* FORMAT
* UCASE
* MID
